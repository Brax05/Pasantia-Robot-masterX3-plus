# Pasantia-Robot-masterX3-plus
Este repositorio contiene el desarrollo, configuración y despliegue del robot Master X3 Plus, potenciado por una NVIDIA Jetson Orin Nano (8GB). El objetivo principal es la integración de un Modelo de Lenguaje Extenso (LLM) ejecutado de manera local para el control autónomo e interacción inteligente con el entorno.
